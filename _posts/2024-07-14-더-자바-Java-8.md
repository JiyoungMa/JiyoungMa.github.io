# '더 자바, Java 8' 강의 후기

> **😉 :** '더 자바, Java 8' 강의 리뷰입니다. <br>
> 강의에 대한 내용 없이, 개인 후기만 있습니다.

오늘 리뷰 할 강의 : [더 자바, Java 8](https://www.inflearn.com/course/the-java-java8/dashboard)

<img width="200" alt="image" src="https://github.com/user-attachments/assets/c65b1f49-104e-40df-a7c8-7a95a3fb2cbc">

이 강의를 들은 이유는 보통 대표적인 면접 질문에 ‘Java8에 추가된 기능이 무엇인가?’라는게 있어서 일단 외우긴 했는데, 기본적으로 자바 개발을 이미 8 이상의 버전으로 시작했었기 때문에, 체감하기가 힘들어서였다.

Java8 이전을 경험해 본 적이 없으니, 달라진 후의 버전이 나에게 너무 익숙했기 때문에 전 후를 비교해서 현재 버전이 어떤게 더 편해졌는지 알기 위해 이 강의를 선택했다.

결론적으로는 내가 원했던 부분을 얻을 수 있어서 매우 만족했다. 총 강의 시간도 5시간 정도라 부담스럽지 않을 것 같다. (난 회사 퇴근하고 와서 듣느라 2주 걸렸지만…)

추가로 항상 바쁘다는 핑계와 쓸 줄 안다는 핑계로 Java 기본서 2권을 충실하게 읽지 않았는데, 2주정도 확보해서 Java 기본서 1,2권을 쭉 읽어야겠다는 생각이 들었다.
